# Portfolio Website

## Getting Started

- Runs on Node version 20
- Run `npm install` to install all dependencies
- Run `npm start` to start the server
- Open `http://localhost:3000` in your browser

## Built With

- Next.js
- Tailwind CSS
- GSAP
- Shadcn UI
- Next-Seo
- Prettier

## For Developers

- Run `npm run dev` to start the development server
- Run `npm run build` to build the project
- Run `npm run start` to start the server
- Run `npm run lint` to lint the project
- Run `npm run format` to format the project

## Maintenance

- Run `npm update` to update all dependencies
- Run `npm audit` to audit all dependencies
- Run `npm audit fix` to fix all vulnerabilities
- Run `npm outdated` to check for outdated dependencies
- Run `npm install <package>@latest` to update a specific package
- Changing contents in the `data` folder will reflect on the website
  - [`faqData.json`](./src/data/faqData.json) for the FAQ section
